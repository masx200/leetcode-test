https://deno.land/#installation

2. 安装 `udd`

https://deno.land/x/udd#installation

#### 使用说明

1. 测试

```
deno task test
```

2.升级依赖

```
deno task udd
```

3.格式化

```
deno task fmt
```

4.lint

```
deno task lint
```
