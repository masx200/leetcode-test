https://deno.land/#installation

2. 安装 `udd`

#### 使用说明

1. 测试

```
deno task test
```

2.升级依赖

```
deno task udd
```

3.格式化

```
deno task fmt
```

4.lint

```
deno task lint
```

5.type check

```
deno task check
```

5.导入模块

```ts
import {} from "https://gitee.com/masx200/leetcode-test/raw/4.1.1/mod.ts";
```


```ts

```

